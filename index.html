NSDC & MIDC Program

Official SSC/Q8303 Syllabus

Our curriculum strictly follows the National Occupational Standards for the Cloud Application Developer qualification. Click any module to expand.

Qualification: Cloud Application Developer
Code: SSC/Q8303 — Level 7
Awarding Body: NSDC (National Skills Development Corporation)
Partner: MIDC (Maharashtra Industrial Development Corp.)
Duration: 6 Months · 300+ Practical Hours

This nationally recognized qualification prepares learners for roles in cloud application development, DevOps engineering, and cloud infrastructure management. Upon completion, graduates receive an NSDC-issued certificate valid across India and recognized by leading IT employers.

01
Module 1: Introduction to Cloud Computing
Foundations · Cloud Models · Deployment Types
▾
  • Introduction to cloud computing concepts and history
  • IaaS, PaaS, and SaaS service models explained
  • Public, private, hybrid, and multi-cloud architectures
  • Major cloud providers overview: AWS, Azure, GCP
02
Module 2: Linux & Networking Fundamentals
OS Basics · Shell · TCP/IP · DNS
▾
  • Linux command line, file system, and user management
  • Shell scripting and automation with Bash
  • TCP/IP networking, subnetting, and routing basics
  • DNS, DHCP, and load balancing concepts
03
Module 3: Application Requirements ★
Core NSDC Module · Architecture · Design Patterns
▾
  • Gathering and analyzing cloud application requirements
  • Microservices vs monolithic architecture decision making
  • Designing for scalability, availability, and fault-tolerance
  • Data storage selection: SQL, NoSQL, object storage
  • Security, compliance, and governance requirements
  • Budgeting and TCO analysis for cloud projects
04
Module 4: Development & Containerization
Git · Docker · Kubernetes · Terraform
▾
  • Git version control, branching strategies and pull requests
  • Docker containerization from Dockerfile to deployment
  • Kubernetes orchestration basics and pod management
  • Infrastructure as Code with Terraform and Ansible
05
Module 5: CI/CD & DevOps Automation
Jenkins · Pipelines · Monitoring
▾
  • Continuous Integration with Jenkins and GitHub Actions
  • Building and managing multi-stage deployment pipelines
  • Automated testing integration in CI/CD workflows
  • Task scheduling with Crontab and event-driven automation
06
Module 6: Service Management ★
Core NSDC Module · ITSM · Monitoring · Incident Response
▾
  • IT Service Management (ITSM) principles and ITIL framework
  • Cloud monitoring with CloudWatch, Grafana, and Prometheus
  • Incident detection, escalation, and resolution workflows
  • SLA management, uptime guarantees, and performance KPIs
  • Capacity planning and auto-scaling strategies
  • Change management and release coordination processes
